## Vanguard - Advanced PHP Login and User Management

- Website: https://vanguardapp.io
- Documentation: https://milos.support-hub.io
- Developed by [Milos Stojanovic](https://mstojanovic.net)

## Install with composer

- $ composer install  or 
- $ composer update

## Set Environment

- $ cp .env.example .env

## Run Migrations

- $ php artisan migrate

## Run seeders

- $ php artisan db:seed

## Default login

- username = admin
- password = admin123
